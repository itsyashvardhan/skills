# PPTX Generation

Create presentation files that are structurally correct, readable, and presentation-ready.

## Workflow

1. Determine audience, purpose, length, tone, and required sections.
2. Build an outline before generating slides.
3. Use consistent slide masters, typography, spacing, colors, and chart styles.
4. Keep each slide focused on one message.
5. Use tables, charts, diagrams, and speaker notes where they improve comprehension.
6. Prefer libraries such as `python-pptx` or `pptxgenjs` when creating or modifying PPTX files.
7. Validate output by checking slide count, text fit, missing media, and broken relationships.

## Quality Bar

- Titles should state the slide takeaway, not just the topic.
- Avoid dense paragraphs; use concise bullets or visual structure.
- Preserve brand assets and existing master layouts when editing an existing deck.
