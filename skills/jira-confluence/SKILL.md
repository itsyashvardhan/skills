# Jira Confluence

Create clear delivery artifacts for planning, execution, and handoff.

## Jira Workflow

1. Capture problem, user value, scope, dependencies, and priority.
2. Write acceptance criteria that are testable.
3. Split work when a ticket mixes unrelated outcomes.
4. Include links, screenshots, logs, or examples when they reduce ambiguity.

## Confluence Workflow

1. Start with purpose, audience, and decision context.
2. Use headings, tables, and status sections for scanability.
3. Separate decisions, open questions, risks, and action items.
