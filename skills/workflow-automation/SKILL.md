# Workflow Automation

Turn repetitive processes into reliable, maintainable automations.

## Workflow

1. Map the current manual process, inputs, outputs, owners, and exceptions.
2. Identify trigger, conditions, actions, and failure handling.
3. Preserve auditability for approvals, client communication, and business records.
4. Choose the simplest durable implementation: script, scheduled job, API integration, or workflow tool.
5. Add validation, logging, and a manual fallback.
6. Document how to run, monitor, disable, and update the automation.
